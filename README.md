# 貸金業務取扱主任者 ○×トレーニング

## iPhoneで使うまで（GitHub Pages）
1. GitHubで新しい公開リポジトリを作成します。
2. このZIPを展開し、中の5ファイルをリポジトリ直下へアップロードします。
3. `Settings` → `Pages` → `Build and deployment` → `Deploy from a branch` を選択します。
4. Branchを `main`、Folderを `/(root)` にして保存します。
5. 数分後に表示されるURLをiPhoneのSafariで開きます。
6. Safariの共有ボタン → `ホーム画面に追加` を選択します。

## 収録機能
- 第1回模試50問
- 回答直後の正誤判定・解説
- 合計点・分野別正答率・弱点分析
- JSONによる問題セットの保存・読込
- 画像・PDFのOCR下書き作成と編集
- PWA対応（ホーム画面起動・基本画面のオフライン利用）

## 注意
OCRは問題文の下書きを作る機能です。正答・解説の自動生成には誤りの可能性があるため、公式解答や監修済み資料との照合が必要です。
