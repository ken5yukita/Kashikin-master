# 貸金主任者 Study v1

## 収録内容
- 4肢択一 2,000問（120個の法令・実務ファクトから生成したオリジナル類題）
- 20分野
- 分野別、ランダム、今日のおすすめ、本試験形式、間違い復習、ブックマーク
- 本試験形式：50問・120分・27/15/5/3の科目構成
- 各選択肢に誤り箇所の引用、正しい内容、根拠資料
- 端末内保存、オフライン利用、バックアップ

## GitHub Pages公開
1. このフォルダ内のファイルをリポジトリのルートへアップロード
2. Settings → Pages → Deploy from a branch → main / root
3. 発行URLをSafariで開き、共有 → ホーム画面に追加

## 注意
初期データの基準日は2025年4月1日を中心としています。公式問題集ではなく、法令・公式範囲を基にした学習用オリジナル類題です。
