# 貸金主任者 過去問1000（公式過去問版）

## 収録内容
- 日本貸金業協会が公開している第1回〜第20回の試験問題：全1,000問
- 各回の公式正答
- 各回の実際の合格基準点
- 回別過去問（元の問題順）
- 分野別・細分類別の重複なしランダム演習
- 間違い復習、成績保存、全問題の回答一覧

## GitHub Pagesの更新
以前のリポジトリに、このフォルダ内のファイルをすべて上書きしてコミットしてください。
公開後に旧版が表示される場合は、iPhoneのホーム画面から旧アプリを削除し、Safariのページを再読込してから「ホーム画面に追加」し直してください。

## データ保存
成績・出題履歴はiPhoneのSafari内（localStorage）だけに保存されます。ログインや有料APIは使用しません。

## 出典
日本貸金業協会「貸金業務取扱主任者資格試験問題」「試験問題の正答」
https://www.j-fsa.or.jp/chief/qualifying_exam/exam_example/

公式資料には問題別の解説は掲載されていないため、本版は公式正答による判定を表示します。
